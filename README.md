## Installation
```bash
$ git clone git@github.com:Piko-Piko-Pon-Taro/navict-recommender.git
```

## How to use
### Setup
```bash
$ cd navict-recommender
$ sh docker/build.sh
$ sh docker/run.sh
$ sh docker/exec.sh
```
